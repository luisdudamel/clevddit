## Commands

```shell
    # Installation command
    npm i or npm/yarn i

    # Running command
    npm start

    # Build command
    npm run build

```

```shell
    # Testing command
    npm run test

    # Testing watch command
    npm run test:watch

    # Testing coverage command
    npm run test:coverage

    # End To End testing command
    npm run cypress:open
```
